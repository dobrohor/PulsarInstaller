# PulsarIntaller - app for multistep installation of multiple apps

## Where to start:
Create a new WPF application and pull the code from this repository.

## How to use:
Compile and run the application. You will be prompted to create list.json file. This file will contain the list of apps you want to install. You can add as many apps as you want, but make sure to follow the format in the example list.json file.

By clicking the "Start Installation" button, the application will start installing the apps in the order they are listed in the list. Please be sure that template apps are ignored.

Further installation steps will be shown in the window.
